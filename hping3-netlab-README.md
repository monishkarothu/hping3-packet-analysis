# Packet Crafting and Analysis with hping3

Hands-on lab using hping3 and tcpdump to craft and analyze network packets, completed as part of Kennesaw State University's networking coursework (NetLab Lab 17).

## What I Did
- Used hping3 to generate and send custom ICMP packets, observing and documenting the results of each command
- Crafted custom TCP packets with hping3 and captured the resulting traffic using tcpdump
- Analyzed captured packet data to confirm expected behavior at each stage of the exercise

## Skills Demonstrated
- Packet crafting and manipulation using hping3
- Traffic capture and analysis using tcpdump
- Practical understanding of ICMP and TCP packet structure
